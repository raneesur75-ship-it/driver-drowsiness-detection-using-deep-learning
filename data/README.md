# Dataset Information

## Overview
- **Total Images**: 2,845
- **Training**: 2,273 images (80%)
- **Testing**: 572 images (20%)
- **Classes**: 4 (Closed_Eyes, Open_Eyes, No_Yawn, Yawn)

## Class Distribution
| Class | Training | Testing |
|-------|----------|---------|
| Closed_Eyes | 574 | 143 |
| Open_Eyes | 574 | 143 |
| No_Yawn | 551 | 143 |
| Yawn | 574 | 143 |

## Data Collection
- Public datasets from Kaggle
- Self-collected photos and videos
- Video frame extraction using OpenCV
- AI-generated augmentations using Stable Diffusion
